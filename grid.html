grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));

En este ejemplo la propiedad auto-fill determina automaticamente el numero de columnas dependiendo del tamaño del siguiente valor minmax(200px, 1fr) permitiendo agregar una nueva columna siempre que sea posible con el tamaño minimo indicado de 200px.

Cuando tienes pocos elementos y quieres que incluso cuando la pantalla en tan grande que no se puedan llenar los campos con las imágenes, puedes utiliar la propiedad autofit, esto ajustará el contenido para ocupar todo el espacio si ya no tienes elementos para hacerlo.

Modificar el tamaño de los elementos

con Grid es posble definir la ubicación y el tamaño de los elementos mediante grid-column-start: 1; grid-column-end: 3; y grid-row-start: 1; grid-row-end: 3; pero existe tambien una forma abreviada que es grid-column: 2/4; y grid-row: 3/4;, otra forma es indicarle el espacio que debe ocupar y no de donde debe comenzar mediante grid-column-start: span 2;.
