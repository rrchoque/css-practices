Especificidad

La especificidad en CSS es una forma de determinar qué reglas CSS se aplican a un elemento cuando hay múltiples reglas que podrían aplicarse. La especificidad se calcula utilizando un sistema de puntos basado en el tipo de selectores utilizados en una regla CSS.

La especificidad de una regla CSS se puede dividir en cuatro componentes principales, que se cuentan y ponderan de la siguiente manera:

  1. Selectores inline (atributo style en el HTML) tienen la mayor especificidad.
  2. Selectores de ID (#id) tienen una especificidad alta.
  3. Selectores de clase, pseudo-clases (:hover, :nth-child, etc.) y atributos ([type="text"]) tienen una especificidad media.
  4. Selectores de tipo (elementos HTML como div, p, etc.) y pseudo-elementos (::before, ::after) tienen una especificidad baja.

Ejemplos de especificidad

1. Selectores Inline
Este texto es rojo.

El estilo inline tiene una especificidad muy alta y sobrescribirá cualquier otra regla CSS.

2. Selectores de ID
Este texto es azul.

El selector de ID tiene una alta especificidad y sobrescribirá las reglas con clases y selectores de tipo.

3. Selectores de Clase
Este texto es verde.

El selector de clase tiene una especificidad media.

4. Selectores de Tipo

Este texto es negro.

El selector de tipo tiene la especificidad más baja.

Ejemplo combinado

        
    <!DOCTYPE html>
    <html lang="es">
        <head>
            <meta charset="UTF-8" />
            <title>Ejemplo de Especificidad</title>
            <style>
            p {
                color: black; /* 1. especificidad más baja */
            }
            .mi-clase {
                color: green; /* 2. especificidad media */
            }
            #mi-id {
                color: blue; /* 3. especificidad alta */
            }
            </style>
        </head>
        <body>
            <p id="mi-id" class="mi-clase">Este texto es azul.</p>
        </body>
    </html>

En este ejemplo, el texto del párrafo será azul porque el selector de ID (#mi-id) tiene la especificidad más alta, sobrescribiendo tanto al selector de clase (.mi-clase) como al selector de tipo (p).

Especificidad acumulada

También es posible combinar selectores, lo que aumenta la especificidad total. Por ejemplo: 

        
    <!DOCTYPE html>
    <html lang="es">
      <head>
        <meta charset="UTF-8" />
        <title>Ejemplo de Especificidad Acumulada</title>
        <style>>
          div p {
            color: red; /* 0-0-2-0 */
          }
          .container p {
            color: green; /* 0-1-1-0 */
          }
          #container p {
            color: blue; /* 1-0-1-0 */
          }
        </style>
      </head>
      <body>
        <div class="container" id="container">
          <p>Este texto es azul.</p>
        </div>
      </body>
    </html>

En este caso, el texto del párrafo será azul porque el selector #container p tiene la especificidad más alta.
