Los espacios afectan al tamaño de la caja

100px * 100px

El div tiene un espacio interior (padding) de 10px, un borde (border) sólido blue de 5px y un espacio exterior (margin) de 15px. Esto afecta al tamaño final del elemento, teniendo un tamaño final de 130px * 130px, esto puede hacer dificil calcular el tamaño final de nuestros elementos.

Propiedad: box-sizing: border-box

100px * 100px

Cuando cambiamos la propiedad box-sizing de content-box (valor por defecto) a border-box, las propiedades width y height incluyen el padding y el border, permitiendonos tener un mejor control del tamaño de nuestros elementos

Desbordamiento (Overflow)

CSS IS AWESOME

Cuando utilizamos elementos con el modelo de la caja podemos llegar a tener desbordamiento del contenido, como se puede ver en el ejemplo anterior, esto se puede llegar a controlar con las siguientes opciones:

overflow: scroll;

CSS IS AWESOME

Esta propiedad agrega las barras de scroll para poder ver el contenido completo, pero si quieres agregar las barras de scroll es recomendable utilizar la propiedad overflow: auto; porque de otra forma puede ser que en algunos dispositivos se agreguen las barras de scroll, incluso cuando el contenido sí cabe en la caja.

overflow: hidden;

CSS IS AWESOME

Esta propiedad esconde el contenido que se desborda.

text-overflow: ellipsis;

CSS IS AWESOME

Esta propiedad agrega (...) antes de la parte en la que se va a recortar el contenido, quedando mucho mejor esteticamente
